Como chegar

Ubatuba, um local maravilhoso, onde há varias praias para praticar diversos tipos de esporte. Para o surf praias que recebem ondulações tanto de sul, leste, sudeste com ondas cavadas até as mais cheias, que servem para um bom aprendizado. Para o Stand Up Paddle, praticamente todas tem condições para praticar.

Principais acessos via Caraguatatuba (BR-101). Pela Rodovia Oswaldo Cruz. E para quem estiver vindo do Rio de Janeiro, Rodovia BR-101.
